# 🖥️ 오라클 서버 접속 및 터미널 명령어 실행 기록 (Command History)

이 문서는 Windows 환경의 CMD(또는 PowerShell)에서 오라클 클라우드 서버에 접속한 후, 서비스 가동을 위해 실행한 모든 명령어를 상세히 기록합니다.

---

## 1. 서버 원격 접속 (SSH Connection)
내 컴퓨터에서 멀리 떨어진 오라클 서버로 입장하는 단계입니다.

1.  **최초 접속 시도 (사설 IP 실수)**:
    * `ssh -i "key_path" ubuntu@{사설 IP}` -> 타임아웃 발생 (사설 IP는 외부 접속 불가 확인)
2.  **공인 IP를 통한 성공적 접속**:
    ```bash
    # 다운로드한 열쇠(.key) 파일 경로를 포함하여 공인 IP로 접속
    ssh -i "C:\Users\jaimo\OneDrive\Desktop\개발자원\오라클\ssh-key.key" ubuntu@{공인 IP}
    ```

---

## 2. 서버 초기 환경 설정 (System Setup)
갓 태어난 서버에 필요한 기본 도구들을 설치하는 과정입니다.

1.  **패키지 목록 업데이트**: 서버의 앱스토어 목록을 최신화합니다.
    ```bash
    sudo apt update
    ```
2.  **Git 및 Python 설치**:
    ```bash
    sudo apt install git -y
    sudo apt install -y python3-pip
    ```
3.  **크롬 브라우저(Headless) 설치**: 서버에서 웹페이지를 띄우기 위해 필수입니다.
    ```bash
    wget [https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb](https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb)
    sudo apt install -y ./google-chrome-stable_current_amd64.deb
    ```
4.  **Tesseract OCR 설치**: 이미지 형태의 연락처를 해독하기 위한 도구입니다.
    ```bash
    sudo apt install -y tesseract-ocr tesseract-ocr-eng
    ```

## 3. 프로젝트 배포 및 실행 (Deployment)
깃허브의 코드를 가져오고 실행에 필요한 부품들을 채우는 과정입니다.

1.  **코드 복제 및 이동**:
    ```bash
    git clone [https://github.com/jaimo012/jobkorea_crawler](https://github.com/jaimo012/jobkorea_crawler)
    cd jobkorea_crawler
    ls -al  # 파일 목록 확인
    ```
2.  **파이썬 라이브러리 설치**:
    ```bash
    pip3 install -r requirements.txt
    ```

---

## 4. 치명적 에러 해결 기록 (Troubleshooting)
실행 중 마주친 라이브러리 충돌 문제를 해결한 특수 명령어입니다.

1.  **OpenSSL 충돌 해결**: `AttributeError`로 인해 `pip3`조차 먹통이 된 상황에서 강제로 시스템 패키지를 도려내고 재설치했습니다.
    ```bash
    # 1. 고장난 시스템 패키지 삭제
    sudo apt-get remove -y python3-openssl
    
    # 2. 시스템 패키지 재설치 (안정 버전)
    sudo apt-get install -y python3-openssl
    
    # 3. pip를 통해 최신 보안 규격으로 강제 업데이트
    sudo pip3 install --upgrade pyOpenSSL cryptography
    ```

## 5. 실행 및 디버깅 (Run & Debug)
실제 봇을 돌리고 상태를 확인하는 최종 단계입니다.

1.  **봇 실행**:
    ```bash
    python3 main.py
    ```
2.  **코드 수정 후 반영 (Git Pull)**:
    ```bash
    git pull  # 깃허브에서 수정한 최신 코드를 서버로 당겨옴
    ```
3.  **디버깅 파일 확인 (WinSCP 활용)**:
    * 실행 실패 시 생성된 `temp_files/` 안의 스크린샷과 HTML 파일을 WinSCP로 가져와서 분석하였습니다.
